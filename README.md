# Leakage Detection in Water Distribution Networks using a Physics-Aware Neural Network

Official code repository for the research paper: **Leakage Detection in Water Distribution Networks using a Physics-Aware Neural Network and Leakage Localization using a Rule-based System.**

**Authors:** Jash Shah, David Daniels, Heet Shah, Abhijeet Salunke  
*Sardar Patel Institute of Technology, Mumbai, India*

[![Pre-print](https://img.shields.io/badge/Pre--print-Zenodo-blue)](https://zenodo.org/records/18938356)
*Note: The paper is currently submitted for peer-review publishing.*

---

## Overview
This repository contains the implementation of a computationally efficient **Physics-Aware Artificial Neural Network (PA-ANN)** designed to detect and localize leaks in Water Distribution Networks (WDNs). By explicitly embedding the conservation of mass and the Hazen-Williams energy equation into the model's learning objective, the PA-ANN significantly reduces false alarms compared to purely data-driven baselines. 

The repository also includes the code for our topology-informed **Rule-Based Localization System**, as well as scripts to reproduce the data processing and sensitivity analyses discussed in the paper using the LeakDB Hanoi benchmark dataset.

---

## Repository Structure

All source code is located in the `src/` directory. The scripts are broken down into four main categories:

### 1. Data Acquisition & Preprocessing
* `download_raw.py`: Downloads the raw LeakDB scenarios and datasets.
* `make_inp_to_csv.py`: Extracts hydraulic network properties (pipe length, diameter, roughness coefficient) from `.inp` files and saves them as a CSV.
* `load_scenario.py`: Parses and converts a raw scenario directory into a structured, machine-learning-ready Pandas DataFrame.

### 2. Leak Detection Models
* `ann.py`: Implementation and training pipeline for the baseline, purely data-driven Artificial Neural Network.
* `paann.py`: Implementation and training pipeline for the proposed Physics-Aware Artificial Neural Network (PA-ANN), incorporating hydraulic loss functions.

### 3. Leak Localization (Rule-Based System)
* `localise_ann.py`: Executes the rule-based fault localization framework using the predictions generated by the baseline ANN.
* `localise_paann.py`: Executes the rule-based fault localization framework using the more physically consistent predictions generated by the PA-ANN.

### 4. Evaluation & Sensitivity Analysis
* `sensitivity_ann.py`: Conducts sensitivity analysis and robustness checks on the baseline ANN model.
* `sensitivity_paann.py`: Conducts sensitivity analysis on the PA-ANN model to evaluate its stability under varying operational conditions.

---

## Usage Pipeline

To reproduce the methodology described in the paper, run the scripts in the following order:

1. **Get the Data:** Run `download_raw.py` to fetch the benchmark data.
2. **Extract Topology:** Run `make_inp_to_csv.py` to get the physical network parameters.
3. **Format Scenarios:** Run `load_scenario.py` to compile the raw data into DataFrames.
4. **Train Models:** Execute `ann.py` and `paann.py` to train both models and output their detection classifications.
5. **Localize Faults:** Run `localise_paann.py` (or `localise_ann.py`) to isolate the exact node/pipe of the detected leak.
6. **Analyze:** Use the `sensitivity_*.py` scripts to generate the robustness metrics and evaluation tables.
